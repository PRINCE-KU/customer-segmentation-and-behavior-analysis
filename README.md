
# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The objective is to identify customer spending patterns, product preferences, subscription trends, and business opportunities through Python, SQL, and Power BI. :contentReference[oaicite:0]{index=0}

---

## 🎯 Project Objectives

- Analyze customer purchasing behavior.
- Identify high-value customer segments.
- Discover top-performing products and categories.
- Evaluate the impact of discounts and subscriptions.
- Generate actionable business recommendations.
- Build an interactive dashboard for decision-making.

## 🧹 Data Cleaning & Preprocessing

The following preprocessing steps were performed:

- Loaded dataset using Pandas
- Performed exploratory analysis
- Checked and handled missing values
- Imputed missing Review Ratings using category median
- Renamed columns using snake_case convention
- Created age_group feature
- Created purchase frequency metrics
- Removed redundant columns
- Loaded cleaned data into PostgreSQL
